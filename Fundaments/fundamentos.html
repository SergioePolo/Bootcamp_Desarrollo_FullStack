¿Qué es HTML?

Lenguaje de marcado que estructura el contenido de una página web. Basado en etiqutas las cuales pueden ser semánticas o no semánticas.

¡No es un lenguaje de programación!

Etiquetas básicas de HTML

Etiqueta Descripción
h1, h2, h3, h4, h5, h6 Títulos y subtítulos de HTML
p Párrafor
img Etiqueta para mostrar las imágenes (src, alt)
ul, ol Etiquetas para definir listar ordenadas y desordenadas
li Etiqueta para definir un item en la lista
div Divisores de contenido - dar estilos
a Redireccionar un link afuera o dentro de nuestra página

Formularios en HTML

Herramienta fundamental para recopilar datos de un usuario, a través de diferentes campos como:

  • Textos
  • Casillas de texto - áreas de texto
  • Listas desplegables
  • Botones
